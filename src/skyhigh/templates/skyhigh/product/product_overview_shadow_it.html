Skyhigh Discover

Skyhigh’s cloud security software enables users to discover the full range of approved and unapproved cloud services in use across the enterprise, understand the risk of each service, and take steps to encourage the use of low risk services.

{% include 'includes/sidebar_product.html' %}

Gain a Comprehensive View of Cloud Service Use

  • Log Processor: Collects and tokenizes logs from firewalls, web proxies, SIEMs, and log aggregation products.
  • CloudRegistry™: Identifies and classifies thousands of cloud services including ones uncategorized by firewalls.
  • Usage Dashboard: Categorizes all cloud services, summarizes usage statistics, and provides navigation filters.

Understand Your Risk Exposure

  • CloudRisk™: Assigns an objective 1-10 risk rating for each cloud service based on data, user/device, service, business, and legal risk.
  • License Management Dashboard: Visually summarizes cloud service usage from a business perspective, showing usage frequency.
  • Security Dashboard: Provides a dashboard view from a security perspective that summarizes high-risk cloud services.

Control Access to Cloud Services

  • Script Generation: Automatically generates scripts for popular firewalls and web proxies to block selected services.
