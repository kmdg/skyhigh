Skyhigh Analyze

Skyhigh’s SaaS Security solution enables users to analyze their enterprise’s cloud usage to identify anomalies in user behavior, malware downloaded from the cloud, inconsistencies in policy enforcement, license utilization, and usage trends.

{% include 'includes/sidebar_product.html' %}

Identify Policy Inconsistencies and Anomalous Behavior

  • Allow/Deny Analytics: Presents detailed statistics on users, data, and requests allowed and denied for each cloud service.
  • CloudAnomaly™: Identifies anomalies in cloud service usage that are indicative of malware, malicious use, or high-risk activity.

Respond with Actionable Intelligence

  • Malware Tracking: Identifies known malware downloaded from cloud services and provides forensic tracking capabilities to determine exposure.
  • Auto-Alerts: Delivers instantaneous or scheduled email alerts for service events, anomalous usage, and operational events.

Visually Summarize and Benchmark Cloud Trends

  • Trending Analytics: Provides a set of interactive charts to explore usage over time and pivot across multiple facets.
  • CloudCompare™: Provides benchmarks for each cloud service including the percentage of all companies that use the service.
